Vue CLI

1.安装：npm install -g @vue/cli使用全局安装-g(脚手架版本3的安装方式，再拉取脚手架2的版本)

2.拉取脚手架2的模板：npm install @vue/cli-init

3.Vue ClI2初始化项目：vue init webpack my-project

4.Vue CLI3初始化项目：vue create my-project

5.如果安装vue cli失败：npm clean cache --force 所在文件夹C:\Users\Administrator\AppData\Roaming\npm-cache手动删除
