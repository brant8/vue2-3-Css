2.runtime-compiler

代码中，可以有template，因为有compiler可以用于编译template

说明

1.安装本地服务器：npm install --save-dev webpack-dev-server webpack中的webpack可作为选择，如果下列步骤失败

2.根据官网配置webpack-config文件

3.错误提示：[webpack-cli] For using 'serve' command you need to install: 'webpack-dev-server' package.

4.参考此处：https://blog.csdn.net/weixin_45841729/article/details/117712400

5.在jason下价格script自动打开网页

6.手动安装webpack-cli后即可运行

7.运行后任何页面更改index立即更新，需要停止ctrl+c确认，并且npm run build即可更新到硬盘
