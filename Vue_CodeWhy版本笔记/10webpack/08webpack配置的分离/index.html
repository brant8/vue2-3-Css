说明（测试error可能版本问题）

1.webpack.config配置分离部分在Build目录下，根据目的截取不同内容

2.生产环境使用base+dev,编译生成使用base+prod

3.需要安装合并插件：npm install webpack-merge --save-dev

4.在分离出来的部分导入webpack-merge路径，并使用exports=webpackMerge.merge（aa,{bb}）进行合并

5.main.js最后一行可以测试以下页面刷新是否正常（不用run build的情况）
